# 서울시 화재 데이터 분석 프로젝트

## 1. 프로젝트 개요

본 프로젝트는 서울시의 화재 발생 데이터와 다양한 공공 데이터를 결합하여, 화재 발생에 영향을 미치는 요인을 분석하고 시각화하는 것을 목표로 합니다.
주요 분석 내용은 다음과 같습니다.

-   **인구통계학적 요인**: 연령대별 인구, 외국인 인구 비율과 화재 발생 건수 간의 상관관계 분석
-   **건물 특성**: 건물의 건축 연도, 노후도 등이 화재 발생에 미치는 영향 분석
-   **시간적 요인**: 계절별, 요일별 화재 발생 패턴 분석
-   **기타 요인**: 기상 데이터(온도, 습도), 비상소화장치 위치 및 개수 등과의 연관성 탐색

## 2. 프로젝트 구조

```
/
├── .conda/                   # Conda 가상환경 폴더
├── data/                     # 원본 데이터 폴더
│   ├── 전국 소방대상물 현황/
│   ├── 전국 화재 현황/
│   └── ...
├── filtered_data/            # 전처리 및 가공된 데이터 폴더 (.csv)
├── imgs/                     # 분석 과정에서 생성된 시각화 이미지 폴더
├── results/                  # 최종 분석 결과물(이미지, csv, json) 폴더
├── *.ipynb                   # 데이터 분석 및 시각화를 위한 Jupyter Notebook 파일
├── *.py                      # 보조 Python 스크립트
├── *.pptx                    # 프로젝트 계획서 및 결과 보고서
└── readme.md                 # 프로젝트 설명 파일
```

## 3. 주요 분석 내용 및 스크립트

-   **`BdgAgeFireAnalysis.ipynb`**: 건물 연령과 화재 발생 간의 관계 분석
-   **`agegrp_per_fire.ipynb`**: 행정구별 연령대 비율과 화재 발생률 분석
-   **`foreginer_data.ipynb`**: 행정구별 외국인 비율과 화재 발생률 상관관계 분석
-   **`Weather_per_fire.ipynb`**: 날씨(기온, 습도)와 화재 발생 건수 관계 분석
-   **`season_fire.ipynb`, `fire_week_day.ipynb`**: 계절별, 요일별 화재 발생 트렌드 분석
-   **`emeg_fire.ipynb`, `seoulmap.ipynb`**: 서울시 내 비상소화장치 분포 및 효과 분석

## 4. 사용된 주요 라이브러리

-   `pandas`

-   `matplotlib`
-   `seaborn`
-   `scikit-learn`

-   `statsmodels`

## 5. 실행 방법

1.  `.conda` 폴더에 포함된 가상환경을 활성화하거나, `requirements.txt` (필요시 생성)를 통해 필요한 라이브러리를 설치합니다.
2.  `Jupyter Notebook`을 실행합니다.
3.  루트 디렉토리의 `*.ipynb` 파일을 열어 각 셀을 순차적으로 실행하며 분석 과정을 확인할 수 있습니다.
4.  분석 결과는 `results/` 및 `imgs/` 폴더에 저장됩니다.

**참고**: 각 Notebook은 독립적으로 실행 가능하도록 설계되었으나, 일부 데이터는 다른 Notebook에서 생성된 `filtered_data` 내의 CSV 파일을 참조할 수 있습니다.
